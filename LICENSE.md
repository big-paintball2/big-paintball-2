## 🔧 Common Issues and Fixes

**❓ GUI does not show up after executing**  
> Make sure your executor is properly attached to **Roblox** before running the script. On mobile, check that the overlay permission is enabled for the executor.

**❓ Silent aim feels inaccurate**  
> In the GUI, adjust the **pred‑smoothness** and **hitbox** sliders. Higher hitbox values make it easier to hit, while lower values look more natural.  

**❓ ESP labels are missing or flickering**  
> This can happen on low‑FPS devices. Try lowering the FPS cap or disabling ESP when close to many players.  

**❓ All guns unlock fails or loops**  
> Wait until you’re fully loaded into a **Big Paintball 2** match, not the lobby. If it still fails, rejoin the server and re‑execute the script.  

**❓ Big paintball 2 script kill all does nothing**  
> Some servers apply restrictions or anti‑cheat that filter damage events. Join a public server and try again; this feature is also less reliable in ranked‑style modes.  

**❓ Script crashes the executor**  
> Your executor version may be outdated. Update to the latest release and, if using KRNL, re‑run the bootstrapper.  

**❓ Mobile version is laggy**  
> Close background apps, disable unnecessary toggles (like ESP), and make sure your device is not overheating.
